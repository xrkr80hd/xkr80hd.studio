# Compact Blog Channel Hero Design

## Goal

Keep the blog channel cover prominent without letting the hero dominate the first screen.

## Approved Layout

- Use a contained 16:9 hero with a practical maximum height.
- Render the cover across the full hero with `object-fit: cover`.
- Place channel details in a small, dark, translucent panel anchored to the left.
- Keep the label, title, description, breadcrumbs, and back button inside the panel.
- Preserve strong contrast and allow the first blog post to remain visible near the fold.
- On small screens, keep the panel compact and allow it to use most of the hero width.

## Scope

Only the public blog channel hero presentation changes. Blog data, channel image selection, post cards, and admin screens remain unchanged.
